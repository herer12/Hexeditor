# Hexeditor

The goal of this project is a hex editor. Details to come.

Goals:

- [x] Get the Data of a file in hex
- [ ] Edit the Data of a file in hex
- [ ] Save the Data of a file in hex
- [x] To be able to look/save/edit the Data in different formats than hex
- [x] To get information about the data segments of this data format
- [ ] Have a GUI
- [ ] Undo/Redo
- [ ] Offset of the data
- [ ] Navigating the Data
